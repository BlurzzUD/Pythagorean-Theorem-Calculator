# Pythagorean Theorem Calculator

A **responsive, interactive Pythagorean Theorem calculator** built with **HTML, CSS, and JavaScript**. This web app allows you to calculate the missing side of a right triangle by entering any two sides. It also provides a visual representation of the triangle and step-by-step calculations.

## How It Works

1. Enter any **two side lengths** in the input fields (`a`, `b`, or `c`).
2. The calculator automatically computes the missing side.
3. The triangle diagram and side labels update in real-time.
4. Detailed calculation steps appear below the triangle for educational purposes.

**Formula used:**  

\[
a^2 + b^2 = c^2
\]

- `a` and `b` are the legs (shorter sides)
- `c` is the hypotenuse (longest side)

## Usage

1. Open the **live demo** above or download/clone the repository.
2. Select your preferred language from the dropdown menu.
3. Enter values for any two sides of the triangle.
4. View the result and step-by-step solution.
5. Click **Clear All** to reset the inputs.
